## 🚀 How to Install

1. Go to the **[Releases](https://github.com/devilstein1/inssist/releases)** page and download the latest version.
2. Extract the downloaded `.zip` file.
3. Open `chrome://extensions/` in your browser.
4. Enable **Developer mode** (top right).
5. Click **"Load unpacked"** and select the extracted folder.
6. Visit [instagram.com](https://www.instagram.com), log in, and enjoy full premium features!

**Tip:** Watch any YouTube tutorial on "load unpacked Chrome extension" if needed.
