## 🚀 How to Install

1. Go to the **[Releases](https://github.com/devilstein1/inssist/releases)** section of this repo.
2. Download the latest release (usually a `.zip` file).
3. Extract the downloaded file.
4. Open your browser and go to `chrome://extensions/`
5. Enable **Developer mode** (top right corner).
6. Click **"Load unpacked"** and select the extracted folder.
7. Visit [instagram.com](https://www.instagram.com), log in, and enjoy full premium features!
